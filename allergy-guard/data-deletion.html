데이터 삭제 안내

우리아이 급식 앱은 모든 데이터를 사용자의 기기에만 저장하며, 외부 서버에 개인정보를 보관하지 않습니다.

1. 앱 내에서 데이터 삭제

  1. 앱 하단의 설정(톱니바퀴 아이콘)을 탭합니다.
  2. 삭제할 자녀 프로필을 선택합니다.
  3. 자녀 삭제 버튼을 탭하면 해당 프로필의 모든 데이터가 즉시 삭제됩니다.

2. 앱 삭제로 전체 데이터 제거

앱을 기기에서 삭제하면 저장된 모든 데이터(자녀 프로필, 알레르기 설정, 캐시)가 즉시 제거됩니다.

삭제되는 데이터

데이터 유형저장 위치삭제 시점
자녀 이름(닉네임)기기 로컬프로필 삭제 또는 앱 삭제 시 즉시
학교 정보기기 로컬프로필 삭제 또는 앱 삭제 시 즉시
알레르기 항목기기 로컬프로필 삭제 또는 앱 삭제 시 즉시
급식 캐시 데이터기기 로컬앱 삭제 시 즉시

회사 서버에 보관되는 개인정보는 없으므로 별도의 서버 데이터 삭제 절차는 필요하지 않습니다.
